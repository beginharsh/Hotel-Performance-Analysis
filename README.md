# Hotel-Performance-Analysis
End-to-End hotel perfromance analysis using real operation data ( Excel Dashboard )

#This Project analyzes 2 years of hotel operational data to evaluate revenue efficiency, cost structure, and profitability.
The dataset is based on a city-based budget hotel operating under an OTA- driven demand model.

The goal was to convert fully operational data into monthly decision-level insight using Advanced Excel.

## Business Context
-Hotel Type: Independent / Budget City Hotel
-Inventory: 14 Roonms
Sales Channel: OTA_lead (approx. 22% commission)
Analysis Period: Jan 2023 - Dec 2024


## Key Metrics Analyzed
#Total Revenue
Totel Expenses
Total Profit
ADR (Average Daily Rate)
Occupancy %
RevPar (Revenue Per Average Room)
COPR (Cost Per Occupied Room)
OTA Comission Project 
Rooms Sold
Seasonality Trends

## Key Insights:
Average ADR(1507) did not gurantee profitabliyu due to high fixed costs.
Average RevPar (1176) remained below CPOR in several months
OTA commissions (22%) significantly reduced net realizations
Fixed operating costs created loss-making months despite healthy occupancy (78%)
Monthly analysis revealed sesonality and structural cost issues that were not visible in the daily data.
